# 智能屏 X1 Pro 产品技术手册

## 产品概述

智能屏 X1 Pro 是 NovaTech 公司推出的新一代 AI 智能交互终端，专为企业级协作场景设计。该产品搭载自研麒麟 9000S 高性能芯片，集成多模态交互引擎与边缘计算能力，可广泛应用于智慧办公、智能零售、工业巡检等领域。

### 核心特性

- **多模态交互**：支持语音、触控、手势、视觉四通道并发交互，响应延迟低于 50 毫秒
- **边缘 AI 推理**：基于麒麟 9000S 内置 NPU，支持本地运行大语言模型，无需云端依赖
- **企业级安全**：硬件级可信执行环境 (TEE)，支持端到端加密通信与生物特征识别
- **开放生态**：提供 SmartScreen SDK v3.2.1 和 RESTful API，支持第三方应用快速集成

### 产品型号

| 型号 | 屏幕尺寸 | 分辨率 | 适用场景 |
|------|---------|--------|---------|
| X1 Pro-15 | 15.6 英寸 | 4K OLED | 会议室、高管办公室 |
| X1 Pro-21 | 21.5 英寸 | 4K OLED | 大堂、展厅、培训室 |
| X1 Pro-27 | 27 英寸 | 5K Mini-LED | 控制中心、指挥大厅 |

### 目标用户

企业 IT 管理员、系统集成商、智能硬件渠道合作伙伴。建议具备基础的网络部署和设备管理经验。

## 技术规格

### 处理器与内存

- **主芯片**：麒麟 9000S，8 核架构（1×Cortex-X2 @ 3.2GHz + 3×Cortex-A78 @ 2.6GHz + 4×Cortex-A55 @ 2.0GHz）
- **NPU**：双核达芬奇架构，算力 26 TOPS (INT8)
- **GPU**：Mali-G710 MC10，支持 8K 视频解码与实时渲染
- **内存**：16GB LPDDR5X（6400 Mbps），可选 32GB
- **存储**：256GB / 512GB UFS 4.0，支持扩展 NVMe SSD 至 2TB

### 显示与交互

- **屏幕**：15.6 英寸 OLED，分辨率 3840×2160，刷新率 120Hz，峰值亮度 1000 nit
- **触控**：电容式 10 点触控，支持主动式触控笔 (4096 级压感)
- **摄像头**：1200 万像素超广角（120 度 FOV），支持 AI 人脸识别与自动取景
- **音频**：四麦克风阵列（远场拾音 8 米），双扬声器（2×5W），支持定向拾音与回声消除

### 连接与接口

- **无线**：Wi-Fi 7 (802.11be)，蓝牙 5.4，5G NR (Sub-6GHz + mmWave)
- **有线**：2×USB-C (支持 DP Alt Mode 与 PD 3.1 供电)，1×HDMI 2.1 输出，1×RJ45 2.5GbE
- **传感器**：环境光传感器、接近传感器、6 轴 IMU、温湿度传感器

### 软件平台

- **操作系统**：HarmonyOS 5.0，支持 OTA 静默升级
- **AI 框架**：MindSpore 2.3，兼容 ONNX 与 TensorFlow Lite 模型部署
- **管理平台**：NovaTech Device Manager（云端/本地部署可选），支持批量配置、监控告警、远程诊断

### 电源与功耗

- **供电方式**：USB-C PD 3.1 (最高 65W) 或 PoE++ (IEEE 802.3bt Type 4)
- **典型功耗**：28W（正常工作），待机 < 3W
- **备用电池**：内置 5000mAh 锂聚合物电池（可持续工作约 45 分钟，用于安全关机）

## 安装指南

### 环境要求

在安装智能屏 X1 Pro 之前，请确认部署环境满足以下条件：

- **工作温度**：0 摄氏度至 40 摄氏度
- **存储温度**：-20 摄氏度至 60 摄氏度
- **相对湿度**：10% 至 90%（无冷凝）
- **海拔**：最高 3000 米（超过 2000 米时性能可能受限）
- **电源稳定性**：电压波动不超过额定值的 ±10%

### 壁挂安装步骤

1. 使用附赠的定位模板在墙面上标记四个安装孔位
2. 使用直径 6mm 的钻头在标记位置钻孔，深度不小于 40mm
3. 插入膨胀螺栓，将壁挂支架用 M4 螺丝固定在墙面上，扭矩 2.5 N·m
4. 将设备背面的 VESA 安装孔（100×100mm）对准支架卡扣，向下推入直至听到"咔嗒"声
5. 连接电源线和网线，使用线缆管理槽整理走线
6. 按下设备侧面的电源键 3 秒开机，首次启动需约 2 分钟完成初始化

### 桌面/支架安装

- 使用附赠的桌面支架底座，将设备底部滑入底座卡槽
- 调整俯仰角度（-5 度至 +25 度），旋紧两侧的阻尼调节旋钮
- 如需使用 Kensington 安全锁，锁孔位于设备背面右下角

### 首次配置

1. 开机后根据屏幕向导选择系统语言和时区
2. 连接 Wi-Fi 或有线网络，系统将自动检查并下载最新固件
3. 使用 NovaTech 企业账号登录，或创建本地管理员账号
4. 在 Device Manager 中注册设备，生成设备唯一标识码
5. 根据需要安装企业应用套件，配置显示墙纸和自动开关机策略

## 安全警告

**请务必在使用前仔细阅读以下安全警告，不当使用可能导致设备损坏、火灾、电击或人身伤害。**

### 电气安全

- 仅使用产品附赠或 NovaTech 官方认证的电源适配器。使用非认证适配器可能导致设备损坏，相关损失不在保修范围内
- 电源插座必须接地且易于触及，以便在紧急情况下快速断开电源
- 雷雨天气或长时间不使用时，请拔下电源适配器
- 禁止自行拆卸、改装电源适配器或设备内部组件

### 环境安全

- 请勿将设备暴露在雨水、液体飞溅或高湿度环境中。本设备防护等级为 IP54，仅适用于室内环境
- 请勿在易燃易爆气体、粉尘或化学品附近使用本设备
- 保持设备通风口畅通，四周至少保留 10cm 空间用于散热
- 避免阳光直射屏幕超过 2 小时，长时间高温可能导致 OLED 烧屏

### 使用安全

- 屏幕表面为精密光学玻璃，请使用附赠的超细纤维清洁布清洁，禁止使用酒精、丙酮等有机溶剂
- 请勿用尖锐物品按压或敲击屏幕，防止面板损坏
- 设备工作时机身温度可能升至 45 摄氏度，属于正常现象，但请避免长时间皮肤接触
- 请勿让儿童在无成人监护下操作本设备

### 数据安全

- 首次使用前请立即修改默认管理员密码
- 定期导出并妥善保管设备配置备份文件
- 废旧设备处置前，请执行安全擦除（"设置 > 系统 > 恢复出厂设置 > 安全擦除"），确保敏感数据不可恢复
- 建议启用全盘加密和屏幕自动锁定功能

### 应急处理

- 如设备冒烟、发出异味或异常声响，立即断开电源并联系技术支持
- 如显示屏破裂，请勿触碰泄漏的液晶材料，戴防护手套后清理
- 如电池发生鼓包或漏液，请勿继续使用，联系售后服务处理

## 维护保养

### 日常维护

- **清洁周期**：建议每周使用干燥的超细纤维布擦拭屏幕和机身表面
- **深度清洁**：每月使用微湿（仅用水）的无尘布清洁通风口和接口区域，清洁前务必断开所有线缆
- **固件更新**：启用自动更新功能，确保设备始终运行最新固件版本。手动检查路径为"设置 > 系统 > 系统更新"
- **日志检查**：管理员应每月登录 Device Manager 查看设备运行日志，关注异常告警和性能趋势

### 定期检修

| 周期 | 项目 | 操作说明 |
|------|------|---------|
| 每季度 | 散热系统 | 检查通风口是否有积尘，使用压缩空气罐进行清理 |
| 每半年 | 线缆连接 | 检查所有线缆接头是否松动，重新插拔确认接触良好 |
| 每年 | 触摸校准 | 在"设置 > 显示 > 触摸校准"中执行全屏校准 |
| 每两年 | 电池检测 | 联系售后服务进行电池健康度检测，容量低于 80% 建议更换 |

### 故障排查

| 现象 | 可能原因 | 处理方案 |
|------|---------|---------|
| 无法开机 | 电源适配器损坏或接触不良 | 检查适配器指示灯，尝试其他 USB-C 电源 |
| 屏幕闪烁 | 刷新率设置不匹配或驱动异常 | 进入安全模式将刷新率调整为 60Hz，更新显卡驱动 |
| 触摸无响应 | 屏幕表面有污渍或静电积累 | 清洁屏幕表面，断开电源 30 秒后重新开机 |
| 网络频繁断开 | Wi-Fi 信道干扰或信号弱 | 切换至 5GHz 频段或有线连接，检查路由器固件 |
| 设备过热 | 通风受阻或环境温度过高 | 清理通风口，改善环境通风，降低屏幕亮度 |
